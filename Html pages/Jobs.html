Jobs

 

Web Developer

Web Developer is an IT professional who designs, creates, and maintains websites and web applications using programming languages. They work with designers to create user interfaces and collaborate with other professionals to build and test applications.

 

Coding Trainee

A Coding Trainee description typically involves learning and applying medical coding principles to translate patient records into coded data for billing and data analysis.

 

Web Designer

Design how websites look and perform. Sort out elements in the website and placement of elements on each page of a website

 

Your responsibilities in these role:​

  • Write code in one or more programming or scripting languages, such as PHP or JavaScript​.
  • Plan and prototype new applications​.
  • Design the architecture of the components of an application​.
  • Build and test Application Program Interfaces (APIs) for applications to exchange data​.
  • Stay up to date with new trends and advancements in web development build and maintain databases​.
  • Meet designers, developers and project staff for progress updates​.
